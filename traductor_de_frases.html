📝 Código del Ejemplo

from deep_translator import GoogleTranslator

# Frase a traducir
frase_original = "Hola, ¿cómo estás?"

# Idiomas destino
idiomas = {
    'en': 'Inglés',
    'fr': 'Francés',
    'de': 'Alemán'
}

print(f"Frase original: {frase_original}\n")

for codigo, nombre in idiomas.items():
    traduccion = GoogleTranslator(source='auto', target=codigo).translate(frase_original)
    print(f"{nombre}: {traduccion}")

📚 Glosario de Funciones

deep_translator

libreria que permite traducir texto desde diferentes servicios

items

devuelve una vista de todos los pares clave-valor del diccionario

source

source es el idioma original que se quiere traducir

target

target es el idioma al que se quiere convertir
